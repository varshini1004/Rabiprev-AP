# 🆘 QuXAT Rabiprev – Animal Emergency Response System for Andhra Pradesh

**Verified animal emergency contacts + first‑aid guides for all 26 districts of Andhra Pradesh.**

🔗 **Live:** https://varshini1004.github.io/Rabiprev-AP/

---

## ✨ Features

- ✅ **26 districts** – Verified contacts across all districts, including remote areas
- 📚 **Emergency guides** – Dog bite, injured stray, sick animal, orphaned baby, cruelty
- 📱 **Offline‑first PWA** – Works without internet; installable on home screen
- 📍 **Google Maps** – One tap to navigate to any address
- 📤 **WhatsApp sharing** – Share any contact instantly
- 🌗 **Dark / Light theme** – Persistent preference
- 🔍 **Filter chips** – NGO / Vet / Government

---

## 🛠️ Tech Stack

HTML5 · CSS3 · JavaScript (ES6) · PWA (Service Worker + Manifest) · GitHub Pages
